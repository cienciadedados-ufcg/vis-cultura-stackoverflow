Estamos interessados na relao entre quanto as pessoas de diferentes pases comentam em questes dos outros. A proporo das pessoas do pas que comentou nas questes de outros est medido na varivel comentaram_prop.

Considerando essa varivel, queremos examinar a relao entre ela e o quo hierrquicas so as relaes em um pas (PDI). Queremos tambm levar em conta o quanto as pessoas daquele pas tm acesso Internet (Internet) e qual o tamanho da base de dados que detectamos daquele pas (usuarios).

Examinando essa relao

Analisando a questo acima, solicitado a relao entre quatro variveis: PDI, comentaram_prop, Internet e usuarios. Para isso, a melhor forma de integrar todas elas a partir de um gafico em que no eixo X tenha a varivel PDI e no eixo Y a varivel comentam_prop. A partir da, podemos verificar como essas duas grandezas se relacionam.

dados %>%
   ggplot(aes(
       x = PDI,
       y = comentaram_prop,
       color = Internet,
       size = log10(usuarios)
   )) +
   geom_point(alpha = .6)

dados %>%
   ggplot(aes(
       x = PDI,
       y = comentaram_prop,
       color = Internet,
       size = log10(usuarios)
   )) +
   geom_point(alpha = .6) +
   facet_grid(site ~ ., scales = "free_y")
